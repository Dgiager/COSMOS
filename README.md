# COSMOS

## The Galaxy Is Yours

COSMOS is a mobile-first, worldwide, location-aware outdoor sky exploration game.

**Owner & Developer: Danny Giager**

### Current prototype

- Uses browser geolocation with coordinates retained only in the browser
- Calculates bright stars above the user's horizon
- Includes worldwide demo locations
- Generates local sky missions
- Tracks Stardust, mission completion, and player level locally
- Includes family-oriented outdoor safety guidance
- Requires no paid API or account

### Live deployment

This repository includes a GitHub Pages deployment workflow. After Pages is enabled for GitHub Actions, pushes to `main` deploy automatically.

### Prototype notice

This is an early prototype for evaluation and continued development. Before a public commercial launch aimed at children or families, the project requires formal privacy, child-safety, security, moderation, accessibility, legal, and trademark review.
